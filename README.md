# Serverless Message App

This is a serverless message handling application built with React on the frontend and AWS services (CDK, Lambda, SQS, DynamoDB, S3) on the backend.

## Technologies

- React
- AWS CDK
- AWS Lambda
- AWS SQS
- AWS DynamoDB
- AWS S3
- AWS API Gateway
